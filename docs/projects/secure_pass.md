# Secure Password Manager 🔐

## Overview

Secure Password Manager is a Python-based application designed to help users generate, store, retrieve, and manage passwords locally. The project was developed to explore encryption concepts, secure data storage, and authentication workflows while gaining hands-on experience with cybersecurity fundamentals.

---

## Problem Statement

Managing multiple passwords securely can be challenging. This project demonstrates how encryption and local databases can be combined to create a simple password management system.

---

## Key Features

* Generate strong random passwords
* Store website credentials securely
* Retrieve saved credentials
* Delete stored entries
* Local SQLite database storage
* Encryption-based password protection
